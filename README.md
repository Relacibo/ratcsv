# ratcsv

Minimalistic CSV Editor which makes use of [Ratatui](https://ratatui.rs).

For now the keybinds are fixed and inspired by the Helix keybinds.

## Install

```bash
cargo install --locked ratcsv
```

## Usage

```bash
rat --help
```
